Google Hash Code

Procedimentos

O evento decorrerá no dia 28 de fevereiro de 2019, quinta-feira, das 14:30h às 22h, Instituto Superior Técnico. O GCE organizará uma sessão de apresentação do Google Hash Code, explicação das regras e esclarecimento de dúvidas. Haverá também uma pequena sessão de talks, dada pelos nossos parceiros. Seguidamente, os participantes serão encaminhados para o coffee-break, onde poderão conhecer os nossos parceiros e as suas ofertas. Depois do coffee-break, a competição terá início.

As inscrições abrem dia 24 de fevereiro às 20h. Para participar basta reunir uma equipa, inscreverem-se aqui e selecionar o hub GCE - Instituto Superior Técnico, Lisboa. Para qualquer dúvida consultem as FAQs do evento.
